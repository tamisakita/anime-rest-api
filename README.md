# Manga Project

## Project proposal
Create an API REST to control Anime registration with Spring Boot, hexagonal architecture and Swagger Documentation.

## Technologies
 * Java 18
 * Spring Boot
 * Maven  
 * H2 Data Base
 * Swagger
 * JPA + Hibernate
